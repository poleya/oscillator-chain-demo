# Модель удочки — Модифицированный маятник Капицы

[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)
[![HTML5](https://img.shields.io/badge/HTML5-Canvas-orange.svg)](https://developer.mozilla.org/ru/docs/Web/HTML/Element/canvas)
[![No Dependencies](https://img.shields.io/badge/dependencies-none-brightgreen.svg)]()

Интерактивная вычислительная программа для исследования **модели "удочки"** — модификации классического маятника Капицы, в которой масса закреплена на конце стержня через упругую пружину и совершает вертикальные колебания под действием высокочастотной вибрации.

> **Научный проект**, разработанный в рамках диссертационного исследования по направлению: *методы структурного математического моделирования механических колебательных систем*.

---

## 📖 О программе

Программа реализует численное моделирование нелинейной динамической системы с двумя степенями свободы:
- **φ** — угол отклонения стержня от вертикали.
- **y** — вертикальное смещение массы относительно точки крепления.

В отличие от классического маятника Капицы, где масса скользит вдоль стержня, в данной модели масса колеблется **строго вертикально** в мировой системе координат. Это приводит к качественно иным динамическим эффектам, включая параметрический резонанс и субгармонические режимы вращения.

Вся программа упакована в **один HTML-файл**, не требует установки, серверной части или подключения к интернету после загрузки.

## ✨ Возможности

### 🔬 Научные инструменты
- Интерактивная симуляция в реальном времени с адаптивным шагом интегрирования.

- Автоматическая классификация режимов: синхронное вращение, субгармоники 2-го и 3-го порядка, сверхсинхронные режимы.
- Расчёт критических частот и визуальное определение зон работы системы (безопасные, пограничные, резонансные).
- **Библиотека эффектов**: полное сохранение параметров, уравнений и траекторий для обеспечения воспроизводимости результатов.
- Журнал научных заметок с сохранением в локальном хранилище браузера.

### 📊 Визуализация
- Анимация маятника с пружиной и вектором вынуждающей силы.
- График вертикального смещения `y(t)` с обозначением ограничителей.
- График угловой скорости `φ̇(t)`.
- Спектральная диаграмма с выделенными пиками и критическими частотами.
- Индикатор текущей зоны работы системы и индикатор ударов массы об ограничители.

### ⚙️ Управление
- Настройка всех физических параметров: масса (`m`), жёсткость (`k`), демпфирование (`b`), частота (`ω`), амплитуда (`F₀`), длина стержня (`L`).
- Гибкая система ограничителей движения массы с пресетами (без ограничений, симметричные, только снизу, произвольные).
- Регулировка скорости анимации (от 0.1x до 5.0x) для детального изучения быстрых процессов.

## 🧮 Физическая модель

### Система
Стержень длины **L** вращается вокруг неподвижного шарнира. На конце стержня через упругую пружину жёсткостью **k** закреплена масса **m**. К массе приложена вертикальная вибрация:
```math
F(t) = F_0 \cdot \cos(\omega t)
```

### Уравнения движения (Лагранж II рода)
**Кинетическая энергия:**
```math
T = \frac{1}{2}m(\dot{y}^2 + L^2\dot{\varphi}^2 + 2L\sin(\varphi)\dot{\varphi}\dot{y})
```
**Потенциальная энергия:**
```math
U = -mgL\cos(\varphi) + mgy + \frac{1}{2}ky^2
```
**Система уравнений движения:**
```math
L\ddot{\varphi} + \sin(\varphi)\ddot{y} + g\sin(\varphi) = 0
```
```math
mL\sin(\varphi)\ddot{\varphi} + m\ddot{y} + b\dot{y} + ky + mg = F_0\cos(\omega t) + mL\cos(\varphi)\dot{\varphi}^2
```

### Структурная матрица (метод Л. Г. Шатихина)
В операторной форме система записывается как **A·x = b**, где **x = [φ̈, ÿ]ᵀ**:
```text
| L             sin(φ) | | φ̈ |   | −g·sin(φ)                                          |
| m·L·sin(φ)    m      | | ÿ | = | F₀·cos(ωt) − mg − ky − bẏ − m·L·cos(φ)·φ̇² |
```

## 📚 Использование

### Исследование режимов
- **Синхронное вращение (φ̇ ≈ ω):** Установите ω = 30 рад/с, F₀ = 100 Н, L = 0.5 м. В спектре будет виден чёткий пик на частоте возбуждения.
- **Субгармоническое вращение 2-го порядка (φ̇ ≈ 2ω):** Установите ω = 20 рад/с, F₀ = 150 Н, L = 0.3 м и включите ограничители (y_min = 0, y_max = 0.16 м).
- **Параметрический резонанс:** Установите частоту возбуждения ω ≈ ω₀ = √(k/m). Наблюдайте экспоненциальный рост амплитуды колебаний.

### Сохранение научного результата
1. Добейтесь интересного динамического режима.
2. Нажмите **⏺ Запись эффекта** и подождите 3-5 секунд.
3. Нажмите **⏹ Стоп запись**, затем **💾 Сохранить**.
4. Введите название и научное описание наблюдения.
5. Результат сохранится в **📚 Библиотеке**. Кнопка "📋 Копировать" в библиотеке выгрузит полную спецификацию (уравнения, параметры, траекторию) в буфер обмена для вставки в отчёт или статью.

## ⚙️ Технические детали

- **Интегрирование:** Классический метод Рунге-Кутты 4-го порядка (RK4), точность O(h⁴).
- **Адаптивный шаг:** N = max(100, ⌈ω·Δt·10⌉) шагов на кадр анимации для гарантии разрешения высокочастотных колебаний.
- **Защита от сингулярности:** При |det(A)| < 10⁻⁶ автоматически применяется упрощённое решение.
- **Обработка ограничителей:** Неупругий удар с коэффициентом восстановления 0.3 и жёстким ограничением максимальной скорости |ẏ| ≤ 50 м/с для предотвращения численного взрыва.
- **Спектральный анализ:** Дискретное преобразование Фурье (DFT) с применением окна Хэмминга для минимизации спектральной утечки.
