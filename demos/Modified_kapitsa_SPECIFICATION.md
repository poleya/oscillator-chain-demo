## 1. Назначение

Интерактивный инструмент для визуализации эффекта динамической стабилизации перевёрнутого маятника (эффект Капицы) с упругой радиальной степенью свободы.

**Область применения:**
- Научно-исследовательская работа в нелинейной механике
- Образовательные цели (лекции, лабораторные работы)
- Верификация теоретических моделей колебательных систем
- Демонстрация метода разделения движений Блехмана

**Научная основа:**
- П. Л. Капица (1951) — эффект динамической стабилизации
- И. И. Блехман — метод разделения движений
- Л. Г. Шатихин — структурные матрицы колебательных систем

## 2. Физическая модель

### 2.1 Система

- Стержень, вращающийся вокруг неподвижного шарнира
- Масса m, скользящая вдоль стержня на пружине жёсткостью k
- Вертикальная вибрация: F(t) = F₀·cos(ω·t)

### 2.2 Обобщённые координаты

| Координата | Физический смысл | Единица |
|------------|------------------|---------|
| φ | Угол отклонения от вертикали (φ = 0 — верх) | рад |
| r | Расстояние от шарнира до массы | м |

### 2.3 Уравнения движения (Лагранж II рода)

**Кинетическая энергия:** T = ½m(ṙ² + r²φ̇²)  
**Потенциальная энергия:** U = −mgr·cos(φ) + ½k(r − l₀)²

**Угловое уравнение:**
mr²φ̈ + 2mrṙφ̇ + mgr·sin(φ) = −F₀·r·cos(ωt)·sin(φ)

**Радиальное уравнение:**
mr̈ − mrφ̇² + k(r − l₀) + bṙ + mg·cos(φ) = F₀·cos(ωt)·cos(φ)

### 2.4 Структурная матрица (метод Шатихина)

Система в операторной форме Q·q = F, где q = [φ, r]ᵀ, p = d/dt:

| | φ | r | F(t) |
|---|---|---|---|
| **φ** | mr²p² + 2mrṗ + mgr·cos(φ) | 2mrφ̇p | −r·sin(φ) |
| **r** | −2mrφ̇p − mrφ̇² | mp² + bp + k | −cos(φ) |

### 2.5 Эффективный потенциал

После метода разделения движений:

**U_eff(Φ) = mgr₀·cos(Φ) + F₀²/(4mω²)·sin²(Φ)**

- Гравитационная компонента: mgr₀·cos(Φ) — максимум при Φ=0 (неустойчивость)
- Вибрационная компонента: F₀²/(4mω²)·sin²(Φ) — минимум при Φ=0 (стабилизация)

### 2.6 Критерий устойчивости (Капица)

**F₀² > 2m²gr₀ω²**

## 3. Численные методы

### 3.1 Интегрирование

**Метод:** Классический Рунге-Кутта 4-го порядка (RK4)

**Адаптивный шаг:** N = max(50, ⌈ω·Δt_scaled·5⌉) шагов на кадр

### 3.2 Обработка ограничений

При выходе r за пределы [r_min, r_max]:
- Координата возвращается к границе
- Скорость инвертируется с коэффициентом 0.3 (неупругий удар)

## 4. Функциональные возможности

### 4.1 Режимы работы

| Режим | Описание |
|-------|----------|
| Ручной запуск | Пользователь задаёт параметры |
| Идеальный эксперимент | Автоматическая установка параметров устойчивости |
| Запись траектории | Сохранение данных в JSON |
| Экспорт | Выгрузка параметров, траектории и заметок |

### 4.2 Визуализация

**Основной экран (2D):**
- Анимация маятника с пружиной
- Вектор силы вибрации F(t)
- Ограничители r_min и r_max
- График эффективного потенциала U(φ)

**Осциллограф:**
- Красная линия: мгновенный угол φ(t)
- Голубая толстая линия: усреднённый угол Φ(t)
- Скользящее окно 300 точек (~6 секунд)
- Система координат с осями и подписями

**Полярная карта U(φ,r):**
- Тепловая карта в полярных координатах
- Цветовая шкала: зелёный (минимум) → жёлтый → красный (максимум)
- Интерактивные ползунки параметров
- Индикатор энергии E/E_barrier
- Красная точка текущего состояния

### 4.3 Инструменты анализа

| Инструмент | Назначение |
|------------|------------|
| Осциллограф | Наблюдение быстрых и медленных компонент |
| Полярная карта | Визуализация поверхности потенциала |
| Индикатор энергии | Оценка близости к вылету из ямы |
| Журнал заметок | Фиксация наблюдений |
| Экспорт данных | Анализ в Python/MATLAB |

### 4.4 Управляющие параметры

**Параметры системы:**

| Параметр | Диапазон | По умолчанию | Единица |
|----------|----------|--------------|---------|
| m (масса) | 0.1 — 2.0 | 0.5 | кг |
| k (жёсткость) | 100 — 3000 | 1000 | Н/м |
| b (демпфирование) | 0 — 5.0 | 2.0 | — |
| ω (частота) | 10 — 100 | 30 | рад/с |
| F₀ (амплитуда) | 0 — 200 | 60 | Н |
| l₀ (длина) | 0.3 — 0.8 | 0.65 | м |

**Ограничители:**

| Параметр | Диапазон | По умолчанию | Единица |
|----------|----------|--------------|---------|
| r_min | 0.1 — 0.7 | 0.35 | м |
| r_max | 0.5 — 1.2 | 0.85 | м |

**Начальные условия:**

| Параметр | Диапазон | По умолчанию | Единица |
|----------|----------|--------------|---------|
| φ₀ | -0.5 — 0.5 | 0.15 | рад |
| φ̇₀ | -5 — 5 | 0.0 | рад/с |

**Управление анимацией:**

| Параметр | Диапазон | По умолчанию |
|----------|----------|--------------|
| Скорость | 0.1x — 5.0x | 1.0x |

## 5. Интерфейс пользователя

### 5.1 Структура

```
┌─────────────────────────────────────────┐
│ [Заголовок]    [📝] [?] [⚙] [✕]        │
├─────────────────────────────────────────┤
│  [Анимация маятника]  [График U(φ)]     │
├─────────────────────────────────────────┤
│ [▶ Старт] [⏸ Стоп]                      │
│ [🔄 Идеальный] [📈 Осциллограф] [🎯 Карта]│
│ [⏺ Запись] [💾 Экспорт]                 │
│ [Статус: Готов | Скорость: 1.0x]        │
├─────────────────────────────────────────┤
│ © Елисеев А.В. 2026                     │
└─────────────────────────────────────────┘
~~~

### 5.2 Модальные окна

1. **⚙ Настройки** — параметры системы, ограничители, начальные условия
2. **? О проекте** — научная основа, матрица Шатихина, критерий Капицы
3. **📝 Журнал заметок** — фиксация наблюдений с сохранением в localStorage
4. **📈 Осциллограф** — графики φ(t) и Φ(t)
5. **🎯 Карта U** — полярная карта потенциала с индикатором энергии
6. **💾 Экспорт** — выгрузка данных в JSON
7. **Выход** — подтверждение выхода с финальным логотипом

### 5.3 Синхронизация параметров

Все ползунки параметров (m, ω, F₀, l₀) синхронизированы между панелью настроек и полярной картой.

## 8. Известные ограничения

### 8.1 Физические

1. Линейная пружина (жёсткость k постоянна)
2. Малые деформации
3. Идеальный шарнир (без трения)
4. Одномерная вертикальная вибрация

### 8.2 Вычислительные

1. Погрешность RK4: O(h⁴)
2. При ω > 80 рад/с возможна нагрузка на CPU
3. Скорость анимации ограничена 60 Гц

### 8.3 Визуальные

1. Тепловая карта: 72×30 сегментов
2. Окно осциллографа: 300 точек
3. Возможны проблемы на экранах < 320px
