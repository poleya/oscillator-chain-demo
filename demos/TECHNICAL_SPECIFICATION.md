## 2. НАЗНАЧЕНИЕ И ЦЕЛИ ПРОЕКТА

### 2.1. Назначение
Проект представляет собой серию интерактивных веб-демонстраций вибрационных эффектов в механических системах. ### 2.2. Цели
1. **Научно-образовательная:** наглядная демонстрация вибрационных эффектов (стабилизация, синхронизация, застревание и др.) для студентов, аспирантов и научных сотрудников.
2. **Методологическая:** визуализация синтеза методов Блехмана (разделение движений) и Шатихина (структурные матрицы) — ключевого вклада автора в диссертационное исследование.

## 3. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 3.1. Модуль 1. Маятник Капицы (реализован)

**Физическая модель:**
- Математический маятник длины `l` на вибрирующем подвесе.
- Амплитуда вибрации `a`, частота `ω`.
- Демпфирование `b`.
- Уравнение движения: `θ'' + b·θ' + (g/l - a·ω²·cos(ωt)/l)·sin(θ) = 0`.

**Интерактивные элементы управления:**
- Ползунок «Амплитуда» (a/l): 0 … 0.5, шаг 0.01
- Ползунок «Частота» (ω/ω₀): 5 … 20, шаг 0.5
- Ползунок «Начальный угол» (θ): 0 … π, шаг 0.1
- Кнопка «Старт/Пауза» — запуск/остановка симуляции
- Кнопка «🔄 Перевернуть» — установка маятника в перевёрнутое положение (θ → π - θ)
- Кнопка «↺ Сброс» — возврат к начальным параметрам

**Визуальные компоненты:**

1. **Анимация маятника** (верхний canvas, 180px):
   - Вибрирующее основание (серый прямоугольник).
   - Стержень маятника (чёрная линия).
   - Груз (красный круг).
   - Частота обновления: ~60 FPS.

2. **График эффективного потенциала** (нижний canvas, 180px):
   - Кривая `V_eff(θ) = -cos(θ) - (a²ω²/4)·cos(2θ)`.
   - Ось X: θ с делениями 0, π/2, π, 3π/2, 2π.
   - Ось Y: V_eff (с автоматическим масштабированием и запасом 15%).
   - Красная точка — текущее положение маятника на кривой потенциала.
   - Заголовок: «Эффективный потенциал».

3. **Структурная матрица Шатихина** (HTML-таблица):
   - Столбцы: `X`, `sin(X)`, `sin(2X)`.
   - Строка оператора: `p²`.
   - Ячейка `sin(X)`: коэффициент `-g/l` (красный = неустойчивость, зелёный = устойчивость).
   - Ячейка `sin(2X)`: коэффициент `a²ω²/4l` (динамически обновляется).
   - Цветовая индикация перехода через критический порог `a²ω²/(2gl) = 1`.

4. **Индикатор устойчивости** (текстовый блок):
   - «✅ УСТОЙЧИВО! Вибрация удерживает маятник вверх ногами!» (зелёный)
   - «⚠️ Неустойчиво (маятник упадёт)» (красный)
   - «✅ Устойчиво (нижнее положение)» (зелёный)

5. **Кнопка справки «?»** (фиксированная, правый верхний угол):
   - Открывает модальное окно с описанием проекта.
   - Содержит: научную основу, инструкцию по использованию, информацию об интеллектуальной поддержке.

6. **Шапка с логотипом:**
   - Две синусоиды (синяя и зелёная) в одной фазе, раздвинутые по вертикали.
   - Красная точка в центре пересечения.
   - Надпись сверху: «механика вибраций».
   - Надпись снизу: «в примерах».

7. **Подвал (footer):**

## 6. НАУЧНАЯ МЕТОДОЛОГИЯ

### 6.1. Синтез методов
Проект демонстрирует **синтез двух методов**:
1. **Метод разделения движений (Блехман)** — разделение на быстрые и медленные движения, осреднение по быстрому времени.
2. **Метод структурных матриц (Шатихин)** — представление системы в виде матрицы связей между переменными.

**Ключевая идея:** оператор осреднения Блехмана действует как **структурный преобразователь** матрицы Шатихина — он аннигилирует внешнее высокочастотное воздействие и генерирует новые внутренние связи (например, `sin(2X)` в маятнике Капицы).

### 6.2. Демонстрируемые эффекты
- **Виброустойчивость** — стабилизация неустойчивого положения равновесия.
- **Структурная бифуркация** — изменение топологии матрицы при переходе через критический параметр.
- **Рождение вибрационных связей** — появление новых внедиагональных элементов в матрице.

## 10. ПРИЛОЖЕНИЯ

### Приложение А. Уравнения маятника Капицы

Полное уравнение движения:
```text
θ'' + b·θ' + (g/l - a·ω²·cos(ωt)/l)·sin(θ) = 0
```

После осреднения по Блехману (метод разделения движений):
```text
X'' + b·X' - (g/l)·sin(X) + (a²ω²/4l)·sin(2X) = 0
```

Эффективный потенциал:
```text
V_eff(X) = -cos(X) - (a²ω²/4l)·cos(2X)
```

Условие стабилизации верхнего положения:
```text
a²ω²/(2gl) > 1
```

### Приложение Б. Структурная матрица

Исходная (до осреднения):
```text
┌────────┬────────┬───────────────┐
│   X    │ sin(X) │     F(t)      │
├────────┼────────┼───────────────┤
│ p²+b·p │  -g/l  │ -a·ω²·cos(ωt) │
└────────┴────────┴───────────────┘
```

Осреднённая (после применения метода Блехмана):
```text
┌────────┬────────┬──────────┐
│   X    │ sin(X) │  sin(2X) │
├────────┼────────┼──────────┤
│ p²+b·p │  -g/l  │ +a²ω²/4l │
└────────┴────────┴──────────┘
```

**Ключевое наблюдение:** внешнее воздействие `F(t)` исчезло, но породило новую внутреннюю связь `sin(2X)`. При `a²ω²/(2gl) > 1` эта связь доминирует, и верхнее положение становится устойчивым.
