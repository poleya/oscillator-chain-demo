🎯 Демонстрации — отдельные физические эффекты

Три самостоятельных приложения, каждое — про один яркий эффект нелинейной динамики: стабилизация перевёрнутого маятника, параметрический резонанс, передача энергии через нелинейные связи.

demos/kapitza-pendulum.html

Маятник Капицы

Классика: стабилизация перевёрнутого маятника вертикальной вибрацией подвеса. Критерий a²ω²/(2gl) > 1. Метод разделения движений И. И. Блехмана, структурные матрицы Шатихина.

Модифицированный маятник Капицы

Масса на пружине вдоль стержня. Две координаты: φ и r. Эффективный потенциал U_eff(φ) = mgr·cos(φ) + F₀²/(4mω²)·sin²(φ). Критерий: F₀² > 2m²gr₀ω². RK4, карта потенциала, осциллограф.

⚡ Быстрый старт

Все приложения — самодостаточные HTML-файлы без зависимостей. Скачайте нужный и откройте в любом современном браузере (Chrome, Firefox, Edge, Safari). Интернет не нужен.

  1. Скачайте oscillator-chain-v2.0-SCIENCE.html из Releases или клонируйте репозиторий.
  2. Откройте файл двойным кликом.
  3. В игровом режиме: постройте цепь, добавьте ровно один возбудитель, выставьте частоту ω.
  4. В научном режиме: «Построить аналитическую АЧХ» → «Валидация» → «Диагностика».
