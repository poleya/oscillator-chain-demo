3-DOF: Полная структурная сеть

📖 О проекте

Разработчик

Елисеев Андрей Владимирович

Докторант, направление: методы структурного математического моделирования механических колебательных систем

Назначение

Интерактивная демонстрация динамических характеристик механических систем с тремя степенями свободы с применением структурных матриц (Л. Г. Шатихин).

Возможности

  • Визуализация динамической податливости системы (амплитудно-частотные характеристики)
  • Анализ рычажных связей между массами (передаточные функции)
  • Интерактивное изменение параметров системы (массы, жёсткости, демпфирование)
  • Отображение структурной матрицы системы в операторной форме
  • Частотное сканирование с автоматическим определением резонансов
  • Экспорт данных для независимой экспертизы и верификации

Экспорт данных (для экспертизы)

Нажмите кнопку 💾 Экспорт в панели управления. Данные будут показаны в специальном окне с возможностью копирования в формате JSON. Файл содержит:

  • Паспорт системы: точные значения всех масс, жесткостей и коэффициентов демпфирования
  • Структурная матрица: операторное представление системы (диагональные элементы Q и связи R)
  • Теоретическая АЧХ: массив точек, рассчитанный аналитически через обратную матрицу импеданса
  • Экспериментальная АЧХ: массив точек, полученных в результате "сканирования"

Эти данные можно импортировать в MATLAB, Python или MathCAD для независимой проверки корректности расчетов.

Как пользоваться

  • Настройте параметры системы через панель "Настройки" (кнопка ⚙)
  • Выберите точку приложения гармонической силы (m₁, m₂ или m₃)
  • Выберите режим отображения: динамическая податливость или рычажные связи
  • Нажмите "▶ Сканировать" для построения АЧХ
  • Используйте "💾 Экспорт" для сохранения результатов
