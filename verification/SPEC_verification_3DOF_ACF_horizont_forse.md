## 2. НАЗНАЧЕНИЕ И ЦЕЛИ

### 2.1. Назначение

Интерактивная веб-демонстрация динамических 
характеристик механических систем с тремя 
степенями свободы (3-DOF) с применением 
структурных матриц Л. Г. Шатихина.

**Программа предназначена для:**
- Визуализации АЧХ колебательных систем
- Анализа рычажных связей между массами
- Демонстрации метода структурных матриц
- Образовательных и научных целей

### 2.2. Цели

1. **Научно-образовательная:** наглядная 
   демонстрация свойств многомерных систем
2. **Методологическая:** визуализация метода 
   структурных матриц Шатихина
3. **Демонстрационная:** использование на 
   конференциях, защитах, в учебных курсах
4. **Верификационная:** экспорт данных для 
   проверки в MATLAB, Python, MathCAD

## 3. ФУНКЦИОНАЛЬНЫЕ ТРЕБОВАНИЯ

### 3.1. Основные функции

#### 3.1.1. Моделирование системы 3-DOF

**Состав системы:**
- Три массы (m₁, m₂, m₃): 1-20 кг
- Две опоры (A и B)
- Межмассовые связи с демпфированием

**Параметры:**
- Жесткости: 0-200 Н/м (опоры), 
  0-100 Н/м (между массами)
- Демпфирование: 0-5 Н·с/м
- Сила: 1-50 Н, гармоническая
- Точка приложения: m₁, m₂ или m₃

#### 3.1.2. Визуализация колебаний

- Три массы (синий, оранжевый, фиолетовый)
- Связи: сплошные (жесткие), 
  пунктирные (демпферы)
- Сила: красная стрелка
- Автомасштабирование амплитуд
- Текущие значения y₁, y₂, y₃

#### 3.1.3. Частотный анализ

**Режимы отображения:**
1. Динамическая податливость y₁, y₂, y₃
2. Рычажная связь |y₂/y₁|
3. Рычажная связь |y₃/y₂|
4. Рычажная связь |y₃/y₁|

**Сканирование:**
- Диапазон: 0.5 - 15 рад/с
- Шаг: 0.3 рад/с
- Установление: 1-20 периодов
- Автоопределение резонансов

#### 3.1.4. Структурная матрица Шатихина

**Содержимое:**
- Диагональ: Qᵢᵢ = mᵢ·p² + Cᵢᵢ·p + Kᵢᵢ
- Вне диагонали: Rᵢⱼ = -(Cᵢⱼ·p + Kᵢⱼ)
- Столбец: F(t)

**Цвета:**
- Синий — диагональ
- Зеленый — связи
- Серый — нулевые
- Красный — силы

**Обновление:** динамическое при 
изменении параметров

#### 3.1.5. Экспорт данных

**Формат:** JSON

**Содержимое:**
1. Метаданные (проект, автор, дата)
2. Паспорт системы (все параметры)
3. Структурная матрица
4. Теоретическая АЧХ (150 точек)
5. Экспериментальная АЧХ

**Способ:** модальное окно с textarea 
и кнопкой копирования

## 6. МАТЕМАТИЧЕСКАЯ МОДЕЛЬ

### 6.1. Уравнения движения

Три дифференциальных уравнения
второго порядка:

**Уравнение 1:**
m₁·y₁'' + C₁₁·y₁' + C₁₂·y₂' +
C₁₃·y₃' + K₁₁·y₁ + K₁₂·y₂ +
K₁₃·y₃ = F₁(t)

**Уравнение 2:**
m₂·y₂'' + C₁₂·y₁' + C₂₂·y₂' +
C₂₃·y₃' + K₁₂·y₁ + K₂₂·y₂ +
K₂₃·y₃ = F₂(t)

**Уравнение 3:**
m₃·y₃'' + C₁₃·y₁' + C₂₃·y₂' +
C₃₃·y₃' + K₁₃·y₁ + K₂₃·y₂ +
K₃₃·y₃ = F₃(t)

Где:
- mᵢ — массы
- Cᵢⱼ — демпфирование
- Kᵢⱼ — жесткость
- Fᵢ(t) — внешние силы

### 6.2. Матрица импеданса

В частотной области:

Z(ω) = K - ω²·M + i·ω·C

Где:
- K — матрица жесткости
- M — матрица масс
- C — матрица демпфирования
- ω — частота

### 6.3. Структурная матрица

Операторная форма 3×4:

Строка y₁:
[Q₁₁] [-R₁₂] [-R₁₃] [-1]

Строка y₂:
[-R₁₂] [Q₂₂] [-R₂₃] [-1]

Строка y₃:
[-R₁₃] [-R₂₃] [Q₃₃] [-1]

Где:
- Qᵢᵢ = mᵢ·p² + Cᵢᵢ·p + Kᵢᵢ
- Rᵢⱼ = Cᵢⱼ·p + Kᵢⱼ

**Особенность:** знаки вне диагонали
противоположны обычной матрице
(по Шатихину)

## 8. ИНСТРУКЦИЯ ПО ИСПОЛЬЗОВАНИЮ

### 8.1. Запуск

1. Откройте файл в браузере
2. Дождитесь splash screen (1.5 с)
3. Приложение готово

### 8.2. Настройка системы

1. Нажмите "Настройки"
2. Установите массы (m₁, m₂, m₃)
3. Настройте связи с опорами A и B
4. Настройте межмассовые связи
5. Установите F₀ и точку приложения
6. Выберите режим графика

### 8.3. Проведение анализа

1. Нажмите "Сканировать"
2. Дождитесь статуса "Готово!"
3. Просмотрите результаты
4. Нажмите "Экспорт" при необходимости

### 8.4. Экспорт данных

1. Нажмите "Экспорт"
2. Нажмите "Копировать"
3. Вставьте в текстовый редактор
4. Сохраните как .json
