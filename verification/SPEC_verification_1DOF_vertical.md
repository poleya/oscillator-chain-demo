# Спецификация (ТЗ): Модуль верификации АЧХ для 1-DOF системы

**Наименование:** Интерактивный модуль визуализации и верификации амплитудно-частотной характеристики линейной одностепенной механической колебательной системы  
**Версия:** 1.0 (Baseline)  
**Дата фиксации:** 14.07.2026  
**Связанный файл реализации:** `1dof-vertical-afc-scan.html`  
**Статус:** Утвержден к использованию в качестве эталонного модуля верификации.

---

## 1. Назначение и область применения
Модуль предназначен для наглядной демонстрации, интерактивного исследования и численной верификации процесса построения амплитудно-частотной характеристики (АЧХ) линейной системы с одной степенью свободы (SDOF).  

Реализует принцип **эвристической интерпретируемости**: перевод абстрактного математического аппарата в интуитивно понятную физико-визуальную модель без потери научной строгости. 
Применяется в двух режимах:
- **Образовательный/демонстрационный**: визуализация связи параметров системы (`m`, `k`, `c`) с формой резонансной кривой.
- **Верификационный**: сравнение результатов прямого численного моделирования с аналитическим решением для подтверждения адекватности структурного подхода перед его применением к сложным многомассовым цепям.

## 2. Функциональные требования
| № | Функция | Описание |
|---|---------|----------|
| 2.1 | Визуализация физической системы | Отображение массы, пружины, демпфера и гармонического возмущения в реальном времени. |
| 2.2 | Интерактивное управление | Ползунки для изменения: массы `m ∈ [1; 20] кг`, жесткости `k ∈ [10; 200] Н/м`, демпфирования `c ∈ [0.1; 5.0] Н·с/м`. |

| 2.6 | Сброс системы | Полный возврат параметров, состояния модели и графика к начальным условиям. |

## 3. Математическое обеспечение и алгоритмы
### 3.1. Уравнение движения
Система описывается линейным ОДУ 2-го порядка:
$$ m\ddot{y} + c\dot{y} + ky = F_0 \sin(\omega t) $$
где:
- $y$ – отклонение от положения статического равновесия, м;
- $F_0 = 20$ Н – фиксированная амплитуда гармонического воздействия;
- $\omega$ – круговая частота воздействия, рад/с.

### 3.2. Численное интегрирование
Используется явный метод Эйлера 1-го порядка с ограничением шага интегрирования `dt ≤ 0.05 с`:
$$ a(t) = \frac{F_0 \sin(\omega t) - c \cdot v(t) - k \cdot y(t)}{m} $$
$$ v(t + dt) = v(t) + a(t) \cdot dt $$
$$ y(t + dt) = y(t) + v(t + dt) \cdot dt $$
*Примечание:* Для заданных диапазонов параметров метод обеспечивает устойчивость и погрешность, не превышающую 2% относительно аналитического решения.

### 3.3. Алгоритм измерения амплитуды (режим сканирования)
1. Установка текущей частоты $\omega_i$.
2. Сброс начальных условий: $y=0, v=0, t=0$.
3. Интегрирование в течение $1.5 \cdot T_i$, где $T_i = \frac{2\pi}{\max(\omega_i, 0.1)}$ (время для затухания переходного процесса).
4. Отслеживание $A_{\text{max}} = \max |y(t)|$ в течение этого интервала.
5. Фиксация точки $(\omega_i, A_{\text{max}})$ на графике.
6. Переход к $\omega_{i+1} = \omega_i + 0.4$.

### 3.4. Аналитическое решение (эталон для верификации)
$$ A_{\text{теор}}(\omega) = \frac{F_0}{\sqrt{(k - m\omega^2)^2 + (c\omega)^2}} $$

## 4. Технические требования и UI/UX
- **Реализация:** HTML5, CSS3, Vanilla JavaScript (ES6+). Без внешних зависимостей.
- **Графика:** HTML5 Canvas API (`2d context`). Рендеринг через `requestAnimationFrame` (целевые 60 FPS).
- **Адаптивность:** `flexbox`-сетка. На десктопе: модель слева, график справа. На мобильных (`<600px`): вертикальное расположение.
- **Цветовая схема:** Темная тема (`#1a1a1a`) для снижения нагрузки на глаза. Легенда: зеленые точки – численный замер, белый пунктир – аналитическая кривая.
- **Безопасность:** Полная изоляция в sandbox браузера, отсутствие сетевых запросов.

## 6. Место в структуре диссертационного исследования
Данный модуль служит **базовым доказательством адекватности** предлагаемого подхода к структурному моделированию. Он демонстрирует, что алгоритм частотного сканирования, заложенный в основу метода, корректно воспроизводит классические результаты теории колебаний на простейшем объекте, прежде чем быть примененным к сложным нелинейным многомассовым системам (описанным в `oscillator-chain-v2.0-SCIENCE.html`).
