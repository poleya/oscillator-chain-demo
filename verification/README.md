# Модули верификации структурных моделей

Данная папка содержит упрощённые, изолированные модули для проверки 
корректности численных методов построения амплитудно-частотных характеристик 
механических колебательных систем.

Каждый модуль:
- Реализован как автономный HTML-файл без внешних зависимостей
- Сопровождается собственной спецификацией (`SPEC_*.md`)
- Работает на мобильных устройствах
- Позволяет визуально сравнить численный замер с аналитическим решением

## 🗺 Roadmap развития

Следующие модули планируется разработать по принципу «база + ветви»:

- **3-DOF цепь** — три массы, три резонансных пика, демонстрация модальных форм колебаний.
- **Нелинейная жёсткость (осциллятор Дуффинга)** — изгиб резонансной кривой, гистерезис, субгармоники.
- **Динамический гаситель колебаний** — специальная настройка 2-DOF системы для подавления вибрации основной массы.
- **Экспорт и анализ** — генерация CSV/JSON, расчёт фазовых траекторий, спектральный анализ (FFT).

## 🎯 Принцип разработки

Все модули верификации сохраняют:
- Единый визуальный стиль (тёмная тема, canvas-графика)
- Единый принцип частотного сканирования
- Единый подход к сравнению численного и аналитического решений
- Адаптивный интерфейс для мобильных устройств

Усложняется только физическая модель и научное содержание.
