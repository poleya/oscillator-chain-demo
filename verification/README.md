# Модули верификации структурных моделей

    Данная папка содержит упрощенные, изолированные модули для проверки 
    корректности численных методов построения АЧХ.

    ## 1dof-vertical-afc-scan.html
    Базовая система с 1 степенью свободы (вертикальные колебания).
    **Научная цель:** Демонстрация принципа эвристической верификации. 
    Численное измерение амплитуды установившихся колебаний при частотном 
    сканировании сравнивается с аналитическим решением в реальном времени.
