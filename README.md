## 🎯 Назначение

Веб-визуализатор, разработанный в рамках докторской диссертации по теме
*«Методы структурного математического моделирования механических колебательных систем»*.

Программный комплекс реализует идеи научной школы **С.В. Елисеева** в интерактивной форме
и предоставляет инструмент для экспериментальной верификации структурного подхода.

## 🔬 Два режима работы

### 🎮 Игровой режим (Game Mode)

Интерактивный симулятор в стиле «Тетрис»: игрок конструирует цепь осцилляторов,
сбрасывая элементы сверху и встраивая их в систему.

**8 типов элементов:**

| № | Элемент | Физический смысл |
|---|---------|------------------|
| 1 | 🚂 Масса (вагон) | Инерционный элемент $m$ |
| 2 | 🟢 Пружина линейная | $F = k \cdot x$ |
| 3 | 🟠 Пружина нелинейная | Модель Дюффинга: $F = kx + \alpha x^3$ |
| 4 | 🔵 Демпфер линейный | $F = -b \cdot v$ |
| 5 | 🟣 Демпфер нелинейный | Квадратичное: $F = -\beta v|v|$ |
| 6 | 🔴 Возбудитель | $F(t) = A\sin(\omega t + \varphi)$ |
| 7 | 🟡 Фазовый сдвиг | Изменение $\varphi$ |
| 8 | 💥 Уничтожитель | Структурное удаление элемента |

**Базовая система:** 1 масса между двумя стойками с двумя пружинами — классический осциллятор.

### 🔬 Научный режим (Science Mode)

Сравнительный анализ двух методов расчёта амплитудно-частотных характеристик (АЧХ):

**Метод 1: Структурный подход (аналитический)**
- Решение алгебраической системы комплексных амплитуд:
  $$(-\omega^2 M + i\omega B + K)X = F$$
- Расчёт динамической податливости $|H(\omega)|$ в [м/Н]
- Диапазон частот: 0.5 — 50 рад/с (≈ 8 Гц)
- **Применим только для линейных систем** ($\alpha = 0$, $\beta = 0$)

**Метод 2: Численный подход (валидация)**
- Интегрирование ОДУ методом Эйлера с 10 подшагами
- Выделение установившегося режима методом наименьших квадратов (МНК)
- **Применим для любых систем** (линейных и нелинейных)

**Диагностика системы:**
- Вывод матриц $M$, $K$, $B$
- Расчёт собственных частот
- Проверка условия резонанса: $b < b_{crit}$

**Ключевой научный результат:**
- Для линейных систем ($\alpha = 0$, $\beta = 0$) оба метода совпадают с ошибкой < 5%
- Для нелинейных систем ($\alpha \neq 0$ или $\beta \neq 0$) аналитический метод недоступен —
  это самостоятельный научный факт, демонстрирующий границы подхода

## 🚀 Как использовать

1. Скачайте файл `oscillator-chain-v2.0-SCIENCE.html` из раздела **Releases**
2. Откройте в любом современном браузере (Chrome, Firefox, Edge, Safari)
3. Работает офлайн — установка не требуется

### Быстрый старт
1. В игровом режиме постройте систему (добавьте массы, пружины, демпферы)
2. Добавьте **ровно один** возбудитель (🔴)
3. Настройте частоту $\omega$ ползунком (шаг 0.1 рад/с)
4. Нажмите 🔬 (научный режим)
5. Нажмите **«Построить аналитическую АЧХ»**
6. Нажмите **«Валидация»** — увидите численные точки
7. Нажмите **«Диагностика»** — изучите параметры системы

## 🔬 Модули верификации (Verification Modules)

Помимо основных демонстрационных и научных модулей, репозиторий содержит 
папку `verification/` с упрощёнными изолированными инструментами для 
проверки корректности численных методов построения АЧХ.

Эти модули служат **эталонным доказательством адекватности** алгоритма 
частотного сканирования перед его применением к сложным многомассовым 
нелинейным системам.

### Текущие модули верификации:

| Файл | Описание | Спецификация |
|------|----------|--------------|
| [`verification/1dof-vertical-afc-scan.html`](verification/1dof-vertical-afc-scan.html) | Линейная система с 1 степенью свободы. Вертикальные колебания массы. Базовая верификация алгоритма частотного сканирования. | [`SPEC_verification_1DOF_vertical.md`](verification/SPEC_verification_1DOF_vertical.md) |
| [`verification/2dof-horizontal-chain-afc-scan.html`](verification/2dof-horizontal-chain-afc-scan.html) | Линейная система с 2 степенями свободы. Горизонтальная цепь, вертикальные колебания. **Силовое** возбуждение массы. Матрица динамической податливости, коэффициент передачи \|y₂/y₁\|. | [`SPEC_verification_2DOF_horizontal.md`](verification/SPEC_verification_2DOF_horizontal.md) |
| [`verification/2dof-horizontal-chain-kinematic-afc-scan.html`](verification/2dof-horizontal-chain-kinematic-afc-scan.html) | Линейная система с 2 степенями свободы. Горизонтальная цепь, вертикальные колебания. **Кинематическое** возбуждение опоры (перемещение). Задача виброзащиты, коэффициент передачи \|y₂/y₁\|. | [`SPEC_verification_2DOF_kinematic.md`](verification/SPEC_verification_2DOF_kinematic.md) |

### Запланированные модули верификации:

- **Нелинейная жёсткость (осциллятор Дуффинга)** — изгиб резонансной кривой, гистерезис, субгармоники.
- **Экспорт и анализ** — генерация CSV/JSON, расчёт фазовых траекторий, спектральный анализ (FFT).
- **Структурный конструктор** — Drag-and-drop сборка схем, автоматическая генерация уравнений состояния.

> **Принцип разработки:** Каждый модуль верификации разрабатывается по модели 
> «база + ветви» — сохраняется единый интерфейс и подход к частотному 
> сканированию, но усложняется физическая модель системы.
