\## 🎯 Назначение / Purpose



Веб-визуализатор, разработанный в рамках докторской диссертации по теме

\*«Методы структурного математического моделирования механических колебательных систем»\*.



Программный комплекс реализует идеи научной школы С.В. Елисеева в интерактивной форме

и предоставляет инструмент для экспериментальной верификации структурного подхода.

\## 🔬 Два режима работы / Two Operating Modes



\### 🎮 Игровой режим (Game Mode)



Интерактивный симулятор в стиле «Тетрис»: игрок конструирует цепь осцилляторов,

сбрасывая элементы сверху и встраивая их в систему.



\*\*8 типов элементов:\*\*



| № | Элемент | Физический смысл |

|---|---------|------------------|

| 1 | 🚂 Масса (вагон) | Инерционный элемент $m$ |

| 2 | 🟢 Пружина линейная | $F = k \\cdot x$ |

| 3 | 🟠 Пружина нелинейная | Модель Дюффинга: $F = kx + \\alpha x^3$ |

| 4 | 🔵 Демпфер линейный | $F = -b \\cdot v$ |

| 5 | 🟣 Демпфер нелинейный | Квадратичное: $F = -\\beta v|v|$ |

| 6 | 🔴 Возбудитель | $F(t) = A\\sin(\\omega t + \\varphi)$ |

| 7 | 🟡 Фазовый сдвиг | Изменение $\\varphi$ |

| 8 | 💥 Уничтожитель | Структурное удаление элемента |



\*\*Возможности:\*\*

\- Настройка параметров в реальном времени (жёсткость, демпфирование, нелинейности)

\- Визуализация сил, натяжения пружин, гармонических воздействий

\- График колебаний масс в реальном времени

\- Адаптивный интерфейс (десктоп + мобильные устройства)



\### 🔬 Научный режим (Science Mode)



Сравнительный анализ двух методов расчёта амплитудно-частотных характеристик (АЧХ):



\*\*Метод 1: Структурный подход\*\*

\- Решение алгебраической системы комплексных амплитуд:

&#x20; $$(-\\omega^2 M + i\\omega B + K)X = F$$

\- Построение матриц $M$, $K$, $B$ для цепи из $N$ осцилляторов

\- Мгновенный результат (без интегрирования)

\- \*\*Применим только для линейных систем\*\* ($\\alpha = 0$, $\\beta = 0$)



\*\*Метод 2: Численный подход\*\*

\- Интегрирование ОДУ методом Эйлера с 8 подшагами

\- Выделение установившегося режима методом наименьших квадратов

\- \*\*Применим для любых систем\*\* (линейных и нелинейных)



\*\*Ключевой научный результат:\*\*

\- Для линейных систем ($\\alpha = 0$, $\\beta = 0$) оба метода совпадают с ошибкой < 5%

\- Для нелинейных систем ($\\alpha \\neq 0$ или $\\beta \\neq 0$) структурный метод неприменим —

&#x20; это самостоятельный научный факт, демонстрирующий границы подхода



\*\*Экспорт данных:\*\*

\- \*\*JSON\*\* — математическая модель (матрицы $M$, $K$, $B$, параметры)

\- \*\*CSV\*\* — АЧХ обоих методов для анализа в Excel / MATLAB / Python

\## 🚀 Как использовать / How to Use



\### Вариант 1: Локальный запуск

1\. Скачайте файл `oscillator-chain-v2.0-SCIENCE.html`

2\. Откройте в любом современном браузере (Chrome, Firefox, Edge, Safari)

3\. Никакой установки не требуется — работает офлайн



\### Вариант 2: Онлайн-демо

Откройте по ссылке GitHub Pages (после публикации):

`https://poleya.github.io/oscillator-chain-demo/oscillator-chain-v2.0-SCIENCE.html`



\### Быстрый старт в научном режиме

1\. Нажмите кнопку 🔬 (вверху слева)

2\. Задайте параметры:

&#x20;  - $N = 3$ (количество масс)

&#x20;  - $m = 1$, $k = 100$, $b = 0.1$

&#x20;  - $\\alpha = 0$, $\\beta = 0$ (линейный случай)

&#x20;  - $A = 10$

3\. Нажмите \*\*«Построить АЧХ (оба метода)»\*\*

4\. Убедитесь, что графики совпадают (ошибка < 5%)

5\. Измените $\\alpha = 10$ — структурный метод станет недоступен

6\. Нажмите \*\*«Экспорт всех данных»\*\* — получите JSON + CSV
